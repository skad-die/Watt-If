# Watt If

An Android app for tenants to calculate monthly electricity bills from submeter kilowatt-hour (kWh) readings and rates, featuring built-in history tracking to monitor power consumption over time.

### Download Pre-Release APK

This is an early testing build. You can download the package directly from this repository:

1. Go to the **[Releases](../../releases)** page.
2. Look for the latest tag marked **Pre-release** and download **`watt-if-v1.0-pre.apk`** from the Assets section.
3. Open the file on your Android device to install it.
